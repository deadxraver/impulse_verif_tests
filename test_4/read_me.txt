Тестовый набор:
1. Исполняемый файл (wrong_hex_converter.elf)
2. Дамп к нему - wrong_hex_converter.elf_nodebug.dump. Main функция в дампе - от pc 0x101a6 до 0x10272
3. Трасса qemu - можно получить при запуске run_elf.sh - включить командную строку "with trace" и выключить "without trace"
4. Командные файлы для запуска программы (с отладчиком и без отладчика):
   
   Запуск командных файлов. 
   Без отладчика:
   # ./run_elf.sh wrong_hex_converter.elf
   С отладчиком (запускаются в разных консолях):
      Сначала (консоль 1):
      # ./run_elf_debug.sh wrong_hex_converter.elf
      Следом (консоль 2): 
      # ./start_gdb_server.sh
   Отладка ведётся в консоли 2.
